# Calculator Web Application

## 📌 Overview
This project is a responsive Calculator Web Application developed using HTML, CSS, and JavaScript. It performs basic mathematical operations through a simple and intuitive user interface.

## 🚀 Features
- Addition
- Subtraction
- Multiplication
- Division
- Clear (AC)
- Delete (DEL)
- Decimal support
- Responsive design

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
